# Legacy Do Not Copy

The legacy project is a reference archive, not a source tree for V2.

Do not copy:

- legacy Next.js app
- legacy FastAPI workflow/services
- legacy DB schema version chain
- legacy adapter mixing patterns
- legacy runtime prompt files as active V2 runtime files

Do not create runtime dependencies on:

- legacy packages or import paths
- legacy environment files or secrets
- legacy migrations
- legacy prompt files
- legacy deployment scripts

Do not use the legacy project to bypass V2 boundaries:

- no row data
- no stored procedure execution
- no DDL/DML apply
- no source apply or deploy
- no raw prompt/provider/SP/row-data/secret persistence
- no Service Codex Runner access to root Development Codex docs or skills

V2 is a new implementation. Legacy material may inform requirements and eval
themes only.
