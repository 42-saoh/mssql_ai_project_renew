# Runtime Roles

| Role | Purpose |
|---|---|
| runtime_sp_analyst | Creates SP analysis proposals from evidence bundles |
| runtime_dependency_analyst | Creates dependency reports from dependency evidence |
| runtime_table_designer | Creates table design previews, never applies DDL |
| runtime_java_mybatis_author | Creates Java/MyBatis draft proposals |
| runtime_reviewer | Checks output policy and review markers |
